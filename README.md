# Лента на подобии ленты ВКонтакте

### Инструкция по запуску:

1. Скачайте приложение с GitHub:

### `git clone https://github.com/Musovvir/team-app.git`

2. Введите в терминале команду:

### `npm i`

3. Затем команду:

### `npm run dev`

### Возможности приложения:

- регистрация
- авторизация
- добавление постов
- изменение контактов
- удаление постов

### Что я использовал:

- React
- Redux
- Redux-Thunk
- Redux-Logger
- React-Router-Dom
- json-server
- ANT Design
- Concurrently
